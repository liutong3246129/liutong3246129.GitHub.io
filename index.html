框架步骤和功能

  • Github建专用仓用于展示个人简历:这是Github提供的功能，以自己用户名Username新建仓名为Username.github.io库，即可展示一页*.md的内容。比如大佬硬件工程师的个人CV
  • Hexo本地框架实现从MD生成index.html网页展示:Hexo是安装在本地的软件/框架，实现本地编辑MD文件，调用Hexo生成静态网页，同步到远程服务器供他人在线访问。
  • Next主题美化:实现定制化功能部分。

Github建专仓

主要参考少数派教程少数派Githubpage实现精简版个人网页展示，
以Github账号名为liutong3246129为例给出具体操作流程

  • 在Github主页点击Repositories的绿色New按钮新建仓库
  • Repository name填写为liutong3246129.github.io点击按钮Creat repositories完成建仓
  • 回到liutong3246129.github.io仓库后点击右上角Setting个性化设置，可以选择Source,Theme,Domain，这里推荐Leap Day主题。

最后在浏览器输入https://liutong3246129.github.io即可看到主题效果。

请注意这里只是展示目录下的*.md文件，如果需要更多的功能，请看下文。

准备工作

  • node环境，可以通过命令行输入node -v检测是否已经安装，如未安装还是可以自行搜索教程(Mac可使用brew install node如果已经安装brew)

我们是为了使用node的包管理器npm(nodepackage manager)来安装Hexo框架

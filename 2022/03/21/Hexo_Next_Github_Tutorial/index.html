Hexo Next Github Tutorial

Posted on 2022-03-21

三种效果

从简洁到完整效果依次实现

  1. Cayman主题Github基础页面
  2. Hexo框架默认主题landscape页面
  3. Hexo框架Nexa主题页面

三个步骤和功能

  1. Github建专仓：用于展示个人简历展示。这是Github提供的功能，以自己用户名Username新建仓名为Username.github.io库，即可展示一页*.md的内容，比如大佬硬件工程师的个人CV。
  2. Hexo本地框架：实现从MD生成index.html网页展示。这里Hexo是安装在本地的软件/框架，用这个框架的好处是可以本地编辑MD文件，调用Hexo生成静态网页，同步到远程服务器，供他人在线访问。
  3. Next主题美化：实现定制化功能。

Github建专仓

主要参考少数派教程Github_page实现精简版个人网页展示，这里以Github账号名为liutong3246129为例给出具体操作流程。

  • 在Github主页点击Repositories的绿色New按钮新建仓库
  • Repository name填写为liutong3246129.github.io点击按钮Creat repositories完成建仓
  • 回到liutong3246129.github.io仓库后点击右上角Setting个性化设置，可以选择Source,Theme,Domain，这里推荐Leap Day主题。

最后在浏览器输入https://liutong3246129.github.io即可看到主题效果。

请注意这里只是展示目录下的*.md文件，如果需要更多的功能，请看下文。
